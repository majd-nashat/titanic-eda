Exploratory Data Analysis on the Titanic dataset using Pandas, Matplotlib and Seaborn
# 🚢 Titanic - Exploratory Data Analysis

## Project Overview
Exploratory Data Analysis on the Titanic dataset to uncover patterns 
that influenced passenger survival.

## Tools & Libraries
- Python
- Pandas
- Matplotlib
- Seaborn

## Key Findings
- Women survived at 74% vs 19% for men
- 1st class passengers had significantly higher survival rates than 3rd class
- Children were prioritized during evacuation

## Dataset
[Titanic Dataset](https://raw.githubusercontent.com/datasciencedojo/datasets/master/titanic.csv)

## Project Structure
- `titanic_eda.py` - Main analysis code
- `titanic_clean.csv` - Cleaned dataset
